# Hyperliquid Spot DCA Bot Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a small Rust CLI (`hl-dca`) that places one spot market-IOC buy on Hyperliquid per invocation, scheduled externally via cron.

**Architecture:** Single binary, stateless, default-deny network selection. Five small modules: `cli` (clap arg parsing), `config` (env + arg resolution), `risk` (pre-flight cap check), `exchange` (Hyperliquid SDK wrapper), and `main` (orchestration). Built on `hyperliquid_rust_sdk` (official) using the `ExchangeClient::order` API with an IOC limit at a slippage-protected price for spot pairs.

**Tech Stack:** Rust 2021, `tokio` (async runtime, required by SDK), `clap` (derive macros), `hyperliquid_rust_sdk` (official), `alloy` (private key parsing — re-exported via SDK examples), `anyhow` (orchestration errors), `thiserror` (typed exchange errors), `time` (UTC timestamp for fill log).

**Note on the Hyperliquid SDK:** `ExchangeClient::market_open` is for **perpetuals**. For **spot**, we use `ExchangeClient::order` with `ClientOrder::Limit` and `tif: "Ioc"`, priced to cross the spread (current_mid * (1 + max_slippage) for buys). This is what "market IOC for spot" actually means in this SDK.

**Note on spot asset naming:** Hyperliquid's spot pairs are *not* keyed by the same string as perps. Task 3 has us run a probe to discover the exact string for the UBTC/USDC pair (and the corresponding key in `all_mids`) before we hardcode it.

```rust
use clap::Parser;

/// CLI args parsed from the command line. Network resolution happens here;
/// secret material (HL_PRIVATE_KEY) is loaded later in `config`.
#[derive(Parser, Debug, Clone)]
#[command(name = "hl-dca", version, about = "Hyperliquid spot DCA buy bot")]
pub struct Args {
    /// USD amount to spend on this single buy.
    #[arg(long)]
    pub usd: f64,

    /// Maximum USD allowed for a single invocation. Required safety guard.
    #[arg(long)]
    pub max_spend_usd: f64,

    /// Spot asset to buy. Only "UBTC" supported in v1.
    #[arg(long, default_value = "UBTC")]
    pub asset: String,

    /// Target Hyperliquid mainnet. Mutually exclusive with --testnet.
    #[arg(long, conflicts_with = "testnet")]
    pub mainnet: bool,

    /// Target Hyperliquid testnet. Mutually exclusive with --mainnet.
    #[arg(long, conflicts_with = "mainnet")]
    pub testnet: bool,

    /// Print what the bot would do and exit; do not place an order.
    #[arg(long)]
    pub dry_run: bool,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum Network {
    Mainnet,
    Testnet,
}

impl Args {
    /// Resolve the chosen network. Errors if neither flag is set.
    pub fn network(&self) -> anyhow::Result<Network> {
        match (self.mainnet, self.testnet) {
            (true, false) => Ok(Network::Mainnet),
            (false, true) => Ok(Network::Testnet),
            (false, false) => Err(anyhow::anyhow!(
                "must pass either --mainnet or --testnet (default-deny)"
            )),
            (true, true) => unreachable!("clap conflicts_with prevents this"),
        }
    }
}
```

```rust
use crate::cli::{Args, Network};
use alloy::signers::local::PrivateKeySigner;
use anyhow::{anyhow, Context};

/// Final, validated runtime configuration. Built once at startup.
pub struct Config {
    pub usd: f64,
    pub max_spend_usd: f64,
    pub asset: String,
    pub network: Network,
    pub dry_run: bool,
    pub wallet: PrivateKeySigner,
}

impl Config {
    /// Build a Config from parsed CLI args plus environment.
    /// Reads `HL_PRIVATE_KEY` from env (the agent wallet's key).
    pub fn from_args(args: Args) -> anyhow::Result<Self> {
        if args.usd <= 0.0 {
            return Err(anyhow!("--usd must be positive (got {})", args.usd));
        }
        if args.max_spend_usd <= 0.0 {
            return Err(anyhow!(
                "--max-spend-usd must be positive (got {})",
                args.max_spend_usd
            ));
        }
        if args.asset != "UBTC" {
            return Err(anyhow!(
                "v1 only supports --asset UBTC (got {:?})",
                args.asset
            ));
        }

        let network = args.network()?;

        let key = std::env::var("HL_PRIVATE_KEY")
            .context("HL_PRIVATE_KEY env var is required (agent wallet private key)")?;
        let key_trimmed = key.trim().trim_start_matches("0x");
        let wallet: PrivateKeySigner = key_trimmed
            .parse()
            .context("HL_PRIVATE_KEY is not a valid hex private key")?;

        Ok(Config {
            usd: args.usd,
            max_spend_usd: args.max_spend_usd,
            asset: args.asset,
            network,
            dry_run: args.dry_run,
            wallet,
        })
    }
}
```

```rust
/// Run all pre-flight risk checks. Called after Config is built and before any
```

```rust
use crate::cli::Network;
use crate::config::Config;
use hyperliquid_rust_sdk::{
    BaseUrl, ClientLimit, ClientOrder, ClientOrderRequest, ExchangeClient, ExchangeDataStatus,
    ExchangeResponseStatus, InfoClient,
};
use std::time::Duration;
use thiserror::Error;

// ┌─ FILLED FROM PROBE in Task 3 ──────────────────────────────────────────┐
// │  The exact strings Hyperliquid uses for the UBTC/USDC spot pair.       │
// │  If these stop working, re-run `cargo run --example probe_spot`        │
// │  against the same network and update.                                  │
// └────────────────────────────────────────────────────────────────────────┘

/// Asset name used in `ClientOrderRequest.asset`. Replace with the value
/// observed in Task 3 (likely "UBTC/USDC" or similar).
const SPOT_ASSET_NAME: &str = "UBTC/USDC";

/// Key under which the mid-price for this pair appears in `info.all_mids()`.
/// Replace with the value observed in Task 3.
const SPOT_MIDS_KEY: &str = "UBTC/USDC";

/// Slippage tolerance: how far above mid we'll cross to fill. 0.005 = 0.5%.
const SLIPPAGE: f64 = 0.005;

/// HTTP timeout for a single round-trip.
const HTTP_TIMEOUT: Duration = Duration::from_secs(10);

#[derive(Debug, Error)]
pub enum ExchangeError {
    #[error("could not build SDK client: {0}")]
    ClientBuild(String),
    #[error("could not fetch mid prices: {0}")]
    Mids(String),
    #[error("mid-price for {0} not found in all_mids")]
    MissingMid(&'static str),
    #[error("mid-price for {0} is not a number: {1}")]
    BadMid(&'static str, String),
    #[error("order rejected by exchange: {0}")]
    Rejected(String),
    #[error("order send returned an ambiguous state: {0}")]
    Ambiguous(String),
    #[error("order send transport error: {0}")]
    Transport(String),
}

/// What we successfully got back from a fill.
#[derive(Debug)]
pub struct Fill {
    pub oid: u64,
    pub avg_px: f64,
    pub size: f64,
}

fn base_url(net: Network) -> BaseUrl {
    match net {
        Network::Mainnet => BaseUrl::Mainnet,
        Network::Testnet => BaseUrl::Testnet,
    }
}

/// Place one spot market-IOC buy of UBTC for the configured USD amount.
/// Steps: fetch current mid, compute size & aggressive limit price, submit
/// IOC limit, return Fill or a typed error.
pub async fn place_spot_market_buy_ubtc(cfg: &Config) -> Result<Fill, ExchangeError> {
    let url = base_url(cfg.network);

    let info = tokio::time::timeout(
        HTTP_TIMEOUT,
        InfoClient::new(None, Some(url.clone())),
    )
    .await
    .map_err(|_| ExchangeError::Transport("InfoClient build timed out".into()))?
    .map_err(|e| ExchangeError::ClientBuild(e.to_string()))?;

    let mids = tokio::time::timeout(HTTP_TIMEOUT, info.all_mids())
        .await
        .map_err(|_| ExchangeError::Transport("all_mids timed out".into()))?
        .map_err(|e| ExchangeError::Mids(e.to_string()))?;

    let mid_str = mids
        .get(SPOT_MIDS_KEY)
        .ok_or(ExchangeError::MissingMid(SPOT_MIDS_KEY))?
        .clone();
    let mid: f64 = mid_str
        .parse()
        .map_err(|_| ExchangeError::BadMid(SPOT_MIDS_KEY, mid_str.clone()))?;

    let limit_px = mid * (1.0 + SLIPPAGE);
    let size = cfg.usd / mid;

    let exchange = tokio::time::timeout(
        HTTP_TIMEOUT,
        ExchangeClient::new(None, cfg.wallet.clone(), Some(url), None, None),
    )
    .await
    .map_err(|_| ExchangeError::Transport("ExchangeClient build timed out".into()))?
    .map_err(|e| ExchangeError::ClientBuild(e.to_string()))?;

    let order = ClientOrderRequest {
        asset: SPOT_ASSET_NAME.to_string(),
        is_buy: true,
        reduce_only: false,
        limit_px,
        sz: size,
        cloid: None,
        order_type: ClientOrder::Limit(ClientLimit {
            tif: "Ioc".to_string(),
        }),
    };

    let response = tokio::time::timeout(HTTP_TIMEOUT, exchange.order(order, None))
        .await
        .map_err(|_| ExchangeError::Ambiguous("order send timed out — check exchange UI".into()))?
        .map_err(|e| ExchangeError::Transport(e.to_string()))?;

    match response {
        ExchangeResponseStatus::Ok(resp) => {
            let data = resp.data.ok_or_else(|| {
                ExchangeError::Ambiguous("OK response with no data field".into())
            })?;
            let status = data.statuses.into_iter().next().ok_or_else(|| {
                ExchangeError::Ambiguous("OK response with empty statuses array".into())
            })?;
            match status {
                ExchangeDataStatus::Filled(filled) => {
                    let avg_px: f64 = filled.avg_px.parse().map_err(|_| {
                        ExchangeError::Ambiguous(format!("avg_px not a number: {}", filled.avg_px))
                    })?;
                    let size: f64 = filled.total_sz.parse().map_err(|_| {
                        ExchangeError::Ambiguous(format!(
                            "total_sz not a number: {}",
                            filled.total_sz
                        ))
                    })?;
                    Ok(Fill {
                        oid: filled.oid,
                        avg_px,
                        size,
                    })
                }
                ExchangeDataStatus::Resting(_) => Err(ExchangeError::Ambiguous(
                    "IOC order returned Resting (should not happen)".into(),
                )),
                ExchangeDataStatus::Error(e) => Err(ExchangeError::Rejected(e)),
                other => Err(ExchangeError::Ambiguous(format!("unexpected status: {other:?}"))),
            }
        }
        ExchangeResponseStatus::Err(e) => Err(ExchangeError::Rejected(e)),
    }
}
```

We do not unit-test this module — its boundary is the SDK, and the testnet smoke test (Task 9) is its real test.

```rust
mod cli;
mod config;
```

```rust
use clap::Parser;
use cli::Args;
use config::Config;
use time::OffsetDateTime;

#[tokio::main]
async fn main() -> std::process::ExitCode {
    match run().await {
        Ok(()) => std::process::ExitCode::SUCCESS,
        Err(e) => {
            // Single-line error to stderr; matches the spec's logging convention.
            eprintln!("error=fatal detail=\"{}\"", oneline(&e.to_string()));
            std::process::ExitCode::from(1)
        }
    }
}

async fn run() -> anyhow::Result<()> {
    let args = Args::parse();
    let cfg = Config::from_args(args)?;
    risk::check(&cfg)?;

    let net = match cfg.network {
        cli::Network::Mainnet => "mainnet",
        cli::Network::Testnet => "testnet",
    };
    let now = OffsetDateTime::now_utc()
        .format(&time::format_description::well_known::Rfc3339)
        .unwrap_or_else(|_| "?".into());

    if cfg.dry_run {
        println!(
            "ts={} mode=dry-run network={} asset={} usd={}",
            now, net, cfg.asset, cfg.usd
        );
        return Ok(());
    }

    let fill = exchange::place_spot_market_buy_ubtc(&cfg)
        .await
        .map_err(|e| anyhow::anyhow!("{e}"))?;

    println!(
        "ts={} network={} asset={} usd={} qty={} avg_px={} oid={}",
        now, net, cfg.asset, cfg.usd, fill.size, fill.avg_px, fill.oid
    );
    Ok(())
}

/// Collapse newlines / quotes so the error line stays single-line and parseable.
fn oneline(s: &str) -> String {
    s.replace('\n', " ").replace('"', "'")
}
```

````markdown
# hl-dca

A small Rust CLI that places one spot market-IOC buy on Hyperliquid per invocation. Schedule it externally to dollar-cost-average into UBTC.

cargo build --release

## One-time setup

1. Fund a Hyperliquid account with USDC. Testnet faucet: https://app.hyperliquid-testnet.xyz/drip
2. In the Hyperliquid UI (API tab), generate an **agent wallet** ("API Wallet"). Record its private key.
3. Approve the agent from your master wallet (UI prompt).
````
