**1. Create `slot_flip_value.dart`** (NEW file):

Create a reusable `SlotFlipValue` StatelessWidget that wraps a formatted value string in a vertical slot-flip animation using `AnimatedSwitcher` + `SlideTransition`.

Key implementation details:
- Constructor takes `required String value` and `TextStyle? style`
- Wrap the entire `AnimatedSwitcher` in `ClipRect` to prevent overflow during slide transition (Pitfall 4)
- Use `GlassCard.shouldReduceMotion(context)` to set duration — `Duration.zero` when reduce-motion is enabled, `const Duration(milliseconds: 250)` otherwise (Pitfall 5)
- `transitionBuilder`: check `animation.status != AnimationStatus.reverse` to distinguish incoming vs outgoing child. Incoming slides from below (`Offset(0, 1)` to `Offset.zero`), outgoing exits upward (`Offset(0, -1)` to `Offset.zero`). Wrap in `SlideTransition` + `FadeTransition` for smooth crossfade (Pitfall 2)
- CRITICAL: Pass `key: ValueKey(value)` to the Text child — this is what triggers the re-animation when the currency format string changes (Pitfall 1)
- `switchInCurve: Curves.easeOut`, `switchOutCurve: Curves.easeIn`

See RESEARCH.md "Code Examples > SlotFlipValue Widget (Complete Implementation)" for the verified full implementation.
